## How to run the app

1. `git clone https://github.com/SFSU-CSC-667/term-project-spring-2017-applejacks.git`
2. `cd term-project-spring-2017-applejacks`
3. `npm install`
4. `node server.js`
5. visit [localhost:3000](localhost:3000)
